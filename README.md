# CFA Clientes API

Prueba técnica – Analista de Desarrollo  
Cooperativa Financiera de Antioquia

Sistema para el **registro, gestión y consulta de clientes** con sus teléfonos y direcciones.

---

# Tecnologías utilizadas

- C#
- .NET
- ASP.NET Web API
- Arquitectura Hexagonal
- PostgreSQL
- Dapper
- Windows Forms
- Postman

---

# Arquitectura del proyecto

El proyecto fue desarrollado utilizando **Arquitectura Hexagonal** para separar las responsabilidades de la aplicación.
